# Backgrounds de Paliers

Placer ici les images de fond pour chaque palier :

## Format requis
- **Format** : JPG (meilleure compression pour photos)
- **Dimensions** : 1080x1920px (portrait mobile) ou 1080x2400px
- **Poids** : < 500KB par image (compression optimisée)
- **Style** : Anime background art, ambiance immersive

## Liste des fichiers nécessaires

- `palier-0-nature.jpg` - Forêt verdoyante, prairie paisible (Hunter x Hunter style)
- `palier-1-fire.jpg` - Flammes, désert ardent, ciel rouge-orangé (Dragon Ball style)
- `palier-2-ocean.jpg` - Océan, vagues, bateau pirate (One Piece style)
- `palier-3-lightning.jpg` - Ciel orageux, éclairs violets, atmosphère électrique (Demon Slayer style)
- `palier-4-forest.jpg` - Forêt dense, bambous, nature luxuriante (Demon Slayer style)
- `palier-5-ice.jpg` - Paysage glacé, cristaux de glace, neige (My Hero Academia style)

## Optimisation
```bash
# Redimensionner et compresser
convert original.jpg -resize 1080x1920^ -gravity center -extent 1080x1920 -quality 85 palier-X-theme.jpg
```

## Sources suggérées
- Anime backgrounds HD (r/AnimeBackgrounds)
- Makoto Shinkai style landscapes
- Studio Ghibli backgrounds
- Unsplash/Pexels avec post-processing anime style

## Effet overlay
Les images seront overlayées avec une couleur semi-transparente pour :
- Améliorer la lisibilité du texte
- Unifier le thème visuel
- Réduire la distraction
