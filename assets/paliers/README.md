# Avatars de Paliers

Placer ici les images des personnages pour chaque palier :

## Format requis
- **Format** : PNG avec transparence
- **Dimensions** : 200x200px minimum (optimisé pour mobile)
- **Poids** : < 200KB par image
- **Style** : Anime art style, visage + épaules

## Liste des fichiers nécessaires

- `palier-0-gon.png` - Gon Freecss (Hunter x Hunter) - Thème Nature/Débutant
- `palier-1-goku.png` - Son Goku (Dragon Ball) - Thème Feu
- `palier-2-luffy.png` - Monkey D. Luffy (One Piece) - Thème Océan
- `palier-3-zenitsu.png` - Zenitsu Agatsuma (Demon Slayer) - Thème Foudre
- `palier-4-tanjiro.png` - Tanjiro Kamado (Demon Slayer) - Thème Forêt
- `palier-5-todoroki.png` - Todoroki Shoto (My Hero Academia) - Thème Glace

## Sources suggérées
- MyAnimeList official art
- Anime screenshots HD
- Fan art libre de droits (DeviantArt, Pixiv avec licence)
