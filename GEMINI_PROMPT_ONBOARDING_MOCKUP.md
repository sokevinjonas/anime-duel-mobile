# Prompt Gemini: Image Mockup Onboarding Page 2

## Instructions pour Gemini Image Generation

Génère une illustration pour l'écran d'onboarding "Comment jouer" d'une app mobile de jeu anime quiz/devinette.

### Style visuel
- Style: Illustration flat design moderne, gamifié, énergique
- Palette: violet mystique (#7C5CE7, #9B7FE8), cyan néon (#00E5CC), rose accent (#FF6B9D)
- Ambiance: joyeuse, dynamique, style anime/gaming

### Composition

**Scène centrale (60% de l'image):**
- Un personnage d'anime style chibi/kawaii au centre avec un point d'interrogation "?" géant au-dessus de sa tête (représentant le mystère)
- Le personnage est flouté/pixelisé pour montrer qu'il est caché
- Autour du personnage: 3-4 bulles de dialogue flottantes contenant des questions courtes:
  - "Cheveux blonds ?" avec une coche verte ✓
  - "A des pouvoirs ?" avec un X rouge
  - "Vient de Naruto ?" avec un "?" orange

**Éléments UI gamifiés (40% de l'image):**
- En haut à droite: icône de Chakra (éclair bleu) avec "8/8"
- En haut à gauche: icône Sharingan (œil rouge) avec "x5"
- En bas: gros bouton violet lumineux avec texte "DEVINER LE PERSO"

**Effets visuels:**
- Particules brillantes/étincelles autour du personnage mystère
- Dégradé de fond violet → bleu sombre
- Ombres douces sous les éléments UI pour effet de profondeur 3D

### Format
- Ratio: 16:9 (paysage horizontal)
- Résolution: 1920x1080px
- Style: illustration vectorielle, couleurs vibrantes, contours nets

### Mots-clés suggérés
anime game UI mockup, quiz guessing game interface, chibi character mystery, violet neon gaming aesthetic, flat design illustration, gamification UI elements, question bubbles, anime guessing game screen

---

## Version alternative (plus simple)

Si la génération complexe échoue, utiliser ce prompt simplifié:

"Create a mobile game UI mockup illustration for an anime guessing game. Center: silhouette of an anime character with a question mark above. Around it: 3 floating question bubbles with checkmarks and X marks. Top corners: game resource icons (lightning bolt, eye icon). Bottom: large glowing purple button. Style: flat design, vibrant purple and cyan colors, energetic gaming aesthetic. 16:9 landscape format."

## Alternative: Utiliser Figma/Canva

Si Gemini ne produit pas un résultat satisfaisant, tu peux aussi créer l'image dans Figma/Canva avec ces éléments:
1. Background: dégradé violet → bleu
2. Personnage chibi anime (pixelisé) au centre avec "?"
3. Bulles de dialogue SVG avec icônes check/X
4. Icônes Chakra et Sharingan en SVG
5. Bouton 3D violet en bas

Exporter en PNG 1920x1080.
